# Audio Atmospherics

Ambient audio layers beneath the broadcast to create atmosphere. The system uses pre-recorded ambient tracks (cold mechanical hum, subtle drone) mixed at -20dB relative to speech.

**Track types**: Intro (3-5s fade-in), Background (loopable ambient bed), Outro (3-5s fade-out), Transition (crossfade between segments)

**Generation**: Pre-recorded WAV files. Future work could include procedurally generated ambient using synth parameters.
